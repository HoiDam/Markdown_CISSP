## Layer 2 / Frame Security Standards

### MACSec / 802.1AE

- Used to secure VPN comms

    Components:

- SecY → Decrypt the frame
- ICV → SecY compute for integrity

    Standards:

- **802.1AR** → specify DevID → device binding
- **802.1AF** → carries key agreement functions → data encryption
